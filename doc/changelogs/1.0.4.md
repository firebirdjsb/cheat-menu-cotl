# Changelog 1.0.4

## UI/UX Modernization

This release focuses on dramatically improving the visual design and user experience of the cheat menu.

### Visual Improvements
- **Modern Color Scheme**: Replaced bland gray UI with vibrant blue accents and better contrast
- **Enhanced Button Styling**: 
  - Active/ON states now use eye-catching blue color
  - Better hover feedback with lighter colors
  - Improved click feedback with darker active states
  - Added padding and margins for cleaner look
- **Title Bar Redesign**: Increased from 20px to 30px with bold blue gradient
- **Visual Indicators**: Added ? (checkmark) and ? (X) icons to toggle buttons for clear ON/OFF states
- **Better Spacing**: Consistent 5px spacing between buttons prevents cluttered appearance
- **Improved Typography**: Bold fonts for important elements, better font sizes throughout

### Layout Improvements
- **Larger Main Window**: Increased from 500x500 to 600x700 pixels for better readability
- **Better Centering**: Menu now properly centers on screen at startup
- **Improved Padding**: Added 5px left padding to all buttons for cleaner alignment
- **Scroll View Optimization**: Better scroll area sizing with 5px padding from edges
- **Dialog Windows**: Ritual and Doctrine selection dialogs increased to 600x650

### User Experience Enhancements
- **Keybind Hints**: Added helpful hint bar at bottom of screen showing:
  - Current keybindings for menu toggle and back navigation
  - Context-aware hints (different when in categories vs main menu)
  - Semi-transparent styling that doesn't obstruct gameplay
- **Better Notifications**: 
  - Repositioned to bottom-center of screen (less intrusive)
  - Added smooth fade-in and fade-out animations
  - Improved visibility with blue-tinted background
  - Better text padding and sizing
- **Category Navigation**: Folder icon (??) and arrow (›) for category buttons
- **Back Button**: Improved back arrow (‹) symbol

### Technical Changes
- **Consistent Styling**: All UI elements now use centralized style functions
- **Better Window Management**: Drag areas properly account for new 30px title bar
- **Toggle Button Spacing**: Added 2px gap between split buttons for clarity
- **Color System**: Established consistent color palette:
  - Background: Dark blue-gray (#1F1F26)
  - Buttons: Medium gray-blue (#40404D)
  - Active/Selected: Vibrant blue (#4D99E6)
  - Title Bar: Medium blue (#3366B3)
  - Notifications: Blue-tinted (#2659A3)

### Bug Fixes
- Fixed notification positioning on various screen resolutions
- Corrected scroll view content width calculation
- Improved button text alignment

## Looking Forward

This update prepares the mod for compatibility with Cult of the Lamb version 1.5.16.1000. 

**Note**: Game assembly updates are required for full compatibility. See `doc/UPGRADE_GUIDE.md` for details on updating the `Assembly-CSharp.dll` reference.
