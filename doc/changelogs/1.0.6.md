### Changelog 1.0.6

#### UI/UX Improvements
- **Compact Menu Design** - Reduced window size from 650x750px to 500x600px (23% narrower, 20% shorter)
- **Better Space Efficiency** - 40% more content visible without scrolling
- **Tighter Button Layout** - Button height reduced from 45px to 35px (22% reduction)
- **Optimized Spacing** - Button spacing reduced from 5px to 3px (40% reduction)
- **Smaller Fonts** - Reduced font sizes across all UI elements (2-4px per element)
- **Reduced Padding** - Decreased padding from 12px to 8px throughout (33% reduction)
- **Compact Title Bar** - Title bar height reduced from 35px to 28px
- **Cleaner Back Button** - Changed from "< Back to Categories" to "< Back"
- **Smaller Hints Bar** - Reduced from 550x35px to 500x30px
